# Exportability model

Exportability is distinct from existence.

A thing can exist, be visible, and still be hard to export, verify, normalize, or replace.

## Exportability classes
- direct_exportable
- exportable_with_limitations
- metadata_only_visible
- human_led_export_required
- provider_blocked
- unknown

## Evaluation axes
- official export path exists
- bulk path exists
- verification path exists
- format openness after export
- collaboration semantics preserved
- replacement target known
