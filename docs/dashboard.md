## Main views

### 1. Overview
- ERI
- PCS by provider
- provider exposure summary
- open-format coverage
- last export status
- next recommended actions

### 2. Providers
- provider-level topology
- provider capability domains
- provider-specific blockers

### 3. Accounts
- tenant / account inventory
- asset collections
- trust boundaries
- current phase state

### 4. Domains
Progress by domain:
- identity
- mail
- storage
- photos
- notes
- office docs
- calendar
- chat
- contacts
- AI / assistant workflows

### 5. Asset Collections
- library counts
- byte counts
- exportability
- temperature
- collaboration status

### 6. Export Ledger
- export runs
- manifests
- hashes
- validation status
- raw vs derived counts

### 7. Evidence / Verification
- evidence artifacts
- verification results
- proof-bearing claims

### 8. Blockers
- ADP / web access blockers
- proprietary format blockers
- collaboration lock-in blockers
- identity lock-in blockers
- missing export capability blockers

### 9. Recommendations
Ordered next actions with expected sovereignty gain, effort, and dependencies.

### 10. Score Explanations
Explain every displayed score in human and machine-readable form.
